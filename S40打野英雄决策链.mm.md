# **S40-打野BP决策链**

## **阵容优先级（4/5L出）**
- 自家战边+软辅
  - 战刺
    - **counter回血、短位移：戈娅艾琳 => 宫本**
    - **多段伤害，且不够灵活 => 铠**
    - **比较灵活，小控制多，有前排适合当跳板 => 杨戬**
    - **比较笨重，爆发低 => 大司命**
- 自家坦边+软辅：
  - AD刺
    - **无控制射手 + 无强控 + 无强制位移 => 镜**
    - **中辅前期弱 + 野区可进 + C好切 => 老虎**
    - **无强制下墙、无加盾回血（保命）、易斩杀 => 暃**
    - **无爆发、无锁头 + 笨重缺带线点 => 韩信**
    - **缺控制、缺爆发 + 软辅抱团好推一片 => 澜**
    - **缺强控、缺爆发 + 偏笨重好拉扯 => 曜**
- 自家战边+硬辅：
  - AP刺
    - **无击落飞行物 / 保命技 + 站位分散 or 脆皮多 => 诸葛**
    - **断大的小控制少 + 站位偏集中 or 法师多 or 怕突脸 => 司马懿**

## **针对射手counter优先级**
- 对面游射+控制多：
  - **公孙离-(宫本、韩信、司马懿)**
  - **敖隐-(宫本、韩信、铠)**
  - **艾琳-(宫本、澜、司马懿)**
  - **戈娅-(宫本、韩信、诸葛)**
- 对面游射+控制少：
  - **孙权-(铠、曜、大司命)**
  - **马可-(镜、铠、宫本)**
  - **孙尚香-(镜、暃、曜)**
  - **苍-(澜、铠、镜)**
  - **守约-(镜、司马懿、宫本)**
  - **李元芳-(镜、韩信、宫本)**
- 呆射+站撸：
  - **狄仁杰-(铠、镜、大司命)**
  - **虞姬-(诸葛、司马懿、铠)**
  - **鲁班-(杨戬、宫本、暃)**
  - **后羿-(大司命、澜、镜)**
  - **蒙犽-(铠、宫本、镜)**
- 呆射+非站撸：
  - **伽罗-(镜、韩信、暃)**
  - **黄忠-(大司命、杨戬、宫本**

## **熟练度优先级**
- AD刺：
  - **韩信 > 曜 > 澜 > 虎 > 镜 > 暃**
- 法刺：
  - **诸葛 > 司马懿**
- 战法刺：
  - **宫本 > 铠 > 大司命 > 杨戬**
