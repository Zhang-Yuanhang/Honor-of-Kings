# **S40 - 打野英雄梯度**
### **基于梯度排序，S40所玩英雄应遵循下述优先级**
- AD刺：镜（无硬控、无强制位移等）、虎（无突脸爆发、前期弱）、暃（无加盾回血、易斩杀）、韩信（整体笨重怕拉扯）、澜（软辅喜欢抱团）、曜（好切C、能带线） 
- AP刺：诸葛（脆皮多，无抵挡飞行物/不可选中/名刀等机制）、司马懿（对面法术伤害多，软辅）
- 战刺：宫本（counter回血、短位移：戈娅艾琳）、铠（多段伤害且不灵活）、大司命 杨戬

### **决策链（基于阵容的决策 - 适合五排-4/5L出）：**
- 战边&软辅 => 拿战刺：偏回复 => 宫本 / 多段伤害 => 铠 / 偏控制+拉扯 => 杨戬 / 笨重+爆发低 => 大司命
- 坦边+软辅：=> 拿AD刺：无控制射+强控+强制位移 => 镜 / 中辅弱+好切C => 老虎 / 灵活、无保命 => 暃 / 笨重，缺带线 => 韩信 / 缺控制、缺爆发、艾琳戈娅 => 澜 / 补位 => 曜
- 战边+硬辅：=> 拿AP刺：偏分散/脆皮多/手长 => 诸葛 / 偏集中/法师多/怕突脸 => 司马懿
